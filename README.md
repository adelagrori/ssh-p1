# Menaxhimi i fjalekalimeve ne Linux
## Pershkrimi i projektit
Ky projekt synon automatizimin e politikave te sigurise per perdoruesit ne sistemet Linux.
## Çfare ben ky script?
* *Politikat e skadimit*: Vendos afatin maksimal per fjalekalimet
* *Paralajmerimi*: Njofton perdoruesin para se te skadoje llogaria
* *Detyrimi i Ndryshimit*: Detiron ndryshimin e fjalekalimit ne hyrjen e pare
## Si perdoret?
1. Jepini lejet: `chmod +x menaxho_fjalekalimet.sh`
2. Ekzekutojeni: `sudo ./menaxho_fjalekalimet.sh`
